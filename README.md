# MinuteOneStory 🎬

Welcome to the code repository for **MinuteOneStory**, a creative writing service specializing in digital storytelling and cinematic scripts.

🔗 **Live Website:** [Click here to view our live portfolio](https://minuteonestory.vercel.app)

## 📖 About This Project
This is the front-end code for our official portfolio website. It serves as a digital storefront for our two-person writing team, showcasing our abilities to write for Reels, YouTube, and full-length screenplays.

## ✨ Features
* **Modern "Midnight" Theme:** Styled with professional dark mode aesthetics.
* **Multi-Page Layout:** Includes Home, Portfolio, About, and Contact pages.
* **Responsive Design:** Looks great on both desktop and mobile phones.
* **Clean URLs:** Optimized routing so links look professional.

## 🛠️ Built With
* **HTML5** - For structure.
* **Tailwind CSS** - For rapid, professional styling without bloated CSS files.
* **Vercel** - For lightning-fast, continuous deployment hosting.
